## Reflection

### 1. Which CSS property gave you the most trouble, and how did you figure it out?

The CSS property that gave me the most trouble was `box-shadow`. I wanted to create a subtle, visually appealing shadow, but I initially struggled to understand how each value affects the final result.Specifically, the horizontal offset, vertical offset, blur radius, spread radius, and color.

I improved my understanding by experimenting with different values in the browser’s developer tools and observing how small adjustments changed the appearance. This hands-on approach helped me better grasp how to control depth and softness in shadows.
